Methods

(static) drawGlyph(data, config, view) → {promise}

Uses the jQuery promise system to try to draw a glyph based on the glyph/sub-glyph of the chosen data object. This does so by requiring the requested glyph type on demand, which, while asynchronous in nature, minimizes the page load time by only requesting/loading the files for the glyphs actually used.
Parameters:
Name Type Description
data object Feature to draw
config object Configuration object meeting the cvit config.json schema
view object CViT view
