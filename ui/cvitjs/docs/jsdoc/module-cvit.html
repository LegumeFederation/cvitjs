Module: cvit

A Javascript version of CViT. cvit.js is the primary module from which most of the work is actually done. Calls for loading initial configuration and data, sets up the canvas for display, then transfers data to request glyphs to be drawn. Note that once paperjs is required in one module, it is added to global scope and should not be required in any other module, at risk of broken functionality.
